# Mini E-commerce Website

A full-stack mini e-commerce web application built with **Django**, **Python**, and **JavaScript**, featuring product listing, shopping cart, checkout, order management, and user authentication.

## Description
This project is a basic e-commerce platform where users can:
- Browse products
- View product details
- Add products to a shopping cart
- Checkout and place orders
- Register and login as a user

Itâ€™s designed for learning purposes and showcases core web development skills.

## User Features
- User registration and login system
- Product listing and details page
- Shopping cart functionality
- Checkout and order processing
- Alerts for successful actions (like adding to cart)
- Simple and clean UI design

## Admin Features
- Admin can manage products, orders, and order items
- Clean Django admin interface

## Tech Stack
- Backend: Python, Django
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Session Management: Django sessions
